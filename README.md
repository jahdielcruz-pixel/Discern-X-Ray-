# Discern-X-Ray

Discern-X-Ray CLI proof of concept

## Installation

Clone the repository and install in editable mode:

```bash
git clone https://github.com/jahdielcruz-pixel/Discern-X-Ray-.git
cd Discern-X-Ray-
pip install -e .
```

## Usage

Run the CLI:

```bash
discernx
```

Or check version:

```bash
discernx --version
```
